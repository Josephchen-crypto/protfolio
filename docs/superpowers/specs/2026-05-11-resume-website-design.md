## 1. 概念与愿景

一个专业级开发者作品集网站，以深色极客风格呈现，通过精致的 GSAP 动效展示技术实力和个人品牌。目标是让面试官在 30 秒内留下"这人很专业"印象。中英文双语切换，服务国内外远程岗位面试。

---

## 2. 设计语言

### 配色

| 角色   | 色值      | 用途           |
| ------ | --------- | -------------- |
| 背景   | `#0a0a0f` | 主背景         |
| 表面   | `#111118` | 卡片背景       |
| 边框   | `#1e1e2e` | 分隔线         |
| 主色调 | `#6366f1` | 强调色（靛蓝） |
| 霓虹紫 | `#a855f7` | 渐变点缀       |
| 霓虹青 | `#22d3ee` | 渐变点缀       |
| 文字主 | `#f8fafc` | 主要文字       |
| 文字次 | `#94a3b8` | 次要文字       |

### 字体

- **标题：** `Space Grotesk` (Google Fonts) — 科技感强
- **正文：** `Inter` — 清晰可读
- **代码：** `JetBrains Mono`

### 动效原则

- **入场动画：** 元素随滚动渐入，带微小上移（translateY: 20px → 0）
- **视差层：** 背景网格/粒子随滚动轻微移动
- **打字效果：** Hero 区域职位标题逐字显示
- **悬浮交互：** 卡片上浮 + 发光边缘
- **页面过渡：** 平滑淡入淡出

---

## 3. 布局与结构

### 单页结构（锚点导航）

1. **Hero** — 全屏开场，姓名 + 职位 + 一句话简介 + CTA 按钮
2. **关于我** — 双语个人简介
3. **工作经历** — 时间线布局
4. **技能** — 标签云 + 熟练度可视化
5. **项目展示** — 卡片网格，带悬浮动效
6. **博客** — 文章列表（静态 Markdown）
7. **联系** — 邮箱 + 社交图标

### 响应式策略

- 移动端：单列布局，导航折叠为汉堡菜单
- 平板：双列卡片
- 桌面：三列卡片

---

## 4. 功能

### 中英文切换

- 顶部固定切换按钮
- 切换时内容平滑过渡
- 语言偏好存储在 localStorage

### 锚点导航

- 顶部固定导航栏，滚动时背景模糊
- 点击平滑滚动到对应区块

### 博客

- 静态 Markdown 文件管理
- 文章列表页 + 详情页
- 支持中英文文章

### 联系

- 邮箱：点击复制
- 社交链接：GitHub / LinkedIn / Twitter 图标

## 7. 内容管理

### 简历内容（中英文）

简历数据存储在 `content/resume/` 下的 JSON 或 Markdown 文件中，支持双语。

### 博客文章

文章以 Markdown 格式存储在 `content/blog/` 目录，使用 front-matter 定义元数据（标题、日期、摘要）。
